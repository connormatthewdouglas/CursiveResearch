# CursiveResearch

Private living research corpus for the CursiveOS team.

## Purpose

This repository collects research notes, sources, competitive intelligence, technical findings, and validation work that inform CursiveOS decisions.

## Confidentiality

Private team material. Do not publish, redistribute, or copy material outside approved project channels without authorization.

## Contribution Notes

- Cite primary sources where possible.
- Include the research date for time-sensitive claims.
- Distinguish verified facts, hypotheses, and recommendations.
