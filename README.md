# CursiveResearch

Private living research corpus for the CursiveOS team.

## Start Here

Read [INDEX.md](INDEX.md) for the chapter library and research status. Read
[methodology/maintaining-the-corpus.md](methodology/maintaining-the-corpus.md)
before adding or restructuring research.

## Purpose

This repository collects source research, technical findings, competitive
intelligence, and validation work that inform CursiveOS decisions. It is
organized so source material remains intact while the team can add curated
analysis over time.

## Repository Layout

| Path | Purpose |
| --- | --- |
| `chapters/` | Human-readable Markdown research chapters organized by topic. |
| `sources/original-docx/` | Preserved original Word documents. Treat these as immutable source masters. |
| `sources/source-register.md` | Provenance, hashes, conversions, and coverage record. |
| `methodology/` | Rules for intake, evidence quality, updates, and review. |
| `tools/` | Repeatable conversion and integrity checking utilities. |

## Confidentiality

Private team material. Do not publish, redistribute, or copy material outside
approved project channels without authorization.

## Current Import Status

The initial eight Word documents were converted into topical Markdown chapters.
The conversion retained source wording and tables; it did not independently
validate time-sensitive or technical claims. Claims should be promoted from
research to decision support only after source verification under the corpus
methodology.
